# YunBaseApp

**YunBaseApp is a base frameword for iOS App writed by objective-c.**

- Platform: iOS 9.0 and later

## Require

- YunKits

- Mantle

## Components

- View

- Theme

- HTTP request

- Error

- Log

- Cache

## Installation

Use the cocoaPods

> `pod 'YunBaseApp'`
